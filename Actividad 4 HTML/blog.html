La importancia de la Semántica HTML

La semántica HTML se refiere al uso de etiquetas que transmiten el significado del contenido, no solo su apariencia. Esto ayuda a los navegadores y tecnologías de asistencia a entender la estructura de la página.

Usar etiquetas como `header`, `main`, `article`, `aside` y `footer` en lugar de simplemente `div` es una buena práctica de accesibilidad.

Beneficios de la Semántica

  • Mejora la **accesibilidad** para usuarios con discapacidades.
  • Optimiza el **SEO** (Search Engine Optimization).
  • Hace el código más legible y **fácil de mantener**.

Introducción a las Tablas HTML

Las tablas HTML son esenciales para mostrar datos tabulares de forma clara. La estructura adecuada es clave para la accesibilidad.

Elementos clave

  1. `table`: El contenedor principal.
  2. `thead`: Agrupa las filas de encabezado.
  3. `tbody`: Agrupa las filas de datos.
  4. `tr`: Representa una fila.
  5. `th`: Define una celda de encabezado.
  6. `td`: Define una celda de datos.
